The Twitter Files

An archive of tweet threads sent by Matt Taibbi (@mtaibbi), Bari Weiss (@bariweiss), and Michael Shellenberger (@ShellenbergerMD). Each file was created by pasting "Embed Tweet" text from Twitter, and depends on the continued existence of the tweets on Twitter for all but the text. At the top of each file is a link to an MHTML file, containing the whole thing with NO reliance on Twitter.
